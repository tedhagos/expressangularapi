

# Client

If you don't have yarn yet, you can get it by npm

```
npm install -g yarn
```

Then

```
cd client
yarn
```

Next, run the local webserver
```
http-server -c-1
```


# Server 

```
yarn
```

Get a command window (terminal), **run the mongo daemon**
```
mongod
```

Get another command window, **run the express server**
```
nodemon server.js
```
